About Us

Hello! I'm Gautam Choudhary currently in final year of Engineering from NSIT, Delhi. In this Project, I have used NLP and Neural Networks to build the model which is able to predict the Movie Reviews (Sentiment Analysis) whether they are positive or negative. And the other application which is Recommender Engine is implemented using Collaborative Filtering Technique. This algorithm at first tries to find similar users based on their activities and preferences and then recommend those movies which other user has seen but you haven't. Also, In this Web Application, I used HTML, CSS, JavaScript to design the front-end and Flask for Back-end. I hope you would like it.
